## Research Themes

### Geometric Algebra (Clifford Algebra)
Coordinate-free mathematical framework enabling unified treatment of rotations, reflections, and projections across dimensions. Applied to:
- Quantum state representation and error correction
- Robotics orientation validation
- Seismic fault slip vector analysis

### Spectral Functionals (λ Family)
Novel spectral analysis methods using FFT-based decomposition:
- **λ_f**: General fault detection functional
- **λ_G**: Power grid stability metric
- **λ(t)**: Time-dependent quantum control functional

### Applications
- **Quantum Computing**: O(n log n) QEC verification vs O(n²) traditional methods
- **Power Systems**: Predictive maintenance and fault detection
- **Fusion Energy**: Plasma instability prediction
- **Robotics**: Quaternion validation (via QTrace platform)
